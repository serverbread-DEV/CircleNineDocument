# CircleNineDocument

简单的galgame小白入门全指南。
认真看完，笨蛋变高手！


## 如何贡献？

首先将此仓库 fork 一份，将 fork 拷贝到你的设备上。

输入命令：
```bash
yarn
npm run docs:dev
```
启动开发模式。

完成你的更改并预览完成后请将你的更改 commit 并推送到你的 fork，然后打开一个 Pull Request。
